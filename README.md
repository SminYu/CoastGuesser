# CoastGuesser

실제 고해상도 해안선 크롭만 보고 세계의 어느 지역인지 맞히는 브라우저 게임입니다.

[이 곳](https://sminyu.github.io/CoastGuesser/)에서 실행 가능합니다.

## 로컬 실행

Python이 설치되어 있다면 프로젝트 폴더에서 다음 명령을 실행하세요.

```powershell
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`을 열면 됩니다.

별도 패키지 설치, API 키, 빌드 과정은 필요하지 않습니다.

## 게임 방식

- 선택한 게임 모드에 따라 전 세계 해안선 전체에서 무작위 지점 출제
- 세계지도 클릭으로 추측 좌표 선택
- 실제 정답까지의 대권거리 계산
- 라운드당 기본 최대 100점, Competitive는 5초 보너스 포함 최대 110점
- 거리 점수 감쇠 기준 5,000km
- 정답 좌표와 거리 공개

## 게임 모드

| 모드 | 제한 시간 | 최대 라운드 | 시간 보너스 |
| --- | ---: | ---: | ---: |
| Classic | 없음 | 10 | 없음 |
| Competitive | 3분 | 15 | 5초 안에 제출 시 +10점 |
| Blitz | 1분 | 10 | 없음 |

## 난이도

| 난이도 | 화면 폭 | 최소 해안 길이 | 육지/바다 최소 비율 | 점수 감쇠 |
| --- | ---: | ---: | ---: | ---: |
| 쉬움 | 1,500km | 300km | 15% | 6,000km |
| 보통 | 1,000km | 100km | 10% | 5,000km |
| 어려움 | 600km | 50km | 5% | 4,000km |

## 해안선 데이터

문제 이미지와 세계지도는 [Natural Earth](https://www.naturalearthdata.com/)의
공개 도메인 `1:50m Coastline` 벡터 자료를 직접 파싱해 렌더링합니다.

- 사전에 정해진 도시나 후보 지역 없음
- 해안선 객체 길이에 `0.3` 지수를 적용한 완화 가중 무작위 추출
- 난이도에 따라 총길이 50~300km 미만의 해안선 객체는 정답 후보에서 제외
- 출제 위도 범위: 남위 65도~북위 65도
- 난이도에 따라 문제 화면 폭 600~1,500km
- 한 게임 내 정답 지점 사이 최소 1,500km 간격
- 최근 40개 출제 지점과 최소 1,000km 간격 유지
- 난이도별로 크롭의 육지/바다 최소 비율을 적용
- 문제 캔버스 내부 해상도 1920×1240
